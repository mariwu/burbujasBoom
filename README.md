## 🫧 Descripción

Juego web interactivo desarrollado con **JavaScript Vanilla** en el que tienes **30 segundos** para explotar el mayor número de burbujas posible. Las burbujas aparecen aleatoriamente en pantalla con colores y posiciones diferentes, y desaparecen solas si no las clicas a tiempo.

Tu puntuación máxima se guarda automáticamente para que puedas superarte en cada partida.

Proyecto desarrollado como parte del módulo **Desarrollo Web en Entorno Cliente (DWEC)** — 2º DAW.

---

## 🎮 ¿Cómo se juega?

1. Pulsa el botón **"Comenzar Juego"**
2. Explota las burbujas que aparecen en pantalla antes de que desaparezcan
3. Tienes **30 segundos** — ¡date prisa!
4. Al finalizar se muestra tu puntuación y se guarda el récord

---

## ✨ Funcionalidades

- ✅ Generación aleatoria de burbujas (posición y color)
- ✅ Temporizador de 30 segundos
- ✅ Puntuación actual y puntuación máxima
- ✅ Récord guardado en **localStorage**
- ✅ Las burbujas desaparecen solas tras 2 segundos
- ✅ Compatibilidad Cross-Browser en gestión de eventos
